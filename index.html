Get started with scribe

with scribe you could convert hand writting text into screen text, create and manage hand written notes, create artistic sketches/drawings and get an overview of your pen precision

Tip 1: on screen writing

read the following tips carefully as they are crucial to the detection system

if your writing is not being detetcted make sure to

  • write at average speed
  • write enlarged and visible
  • follow calibration patterns
  • try changing your starting direction(if it's mistaking letters)
  • do not write with cursive

    • before writing

  • make sure to go through the calibration pattern
  • note : digits and signs are not recognised
  • the two empty btns towards the left are called auxilary keys and you can assign any letter, sign or digit to each one by right clicking on it and typing the desired letter

Tip 2: sketching

the sketch pad can be used to make artistic drawings the opacity tool can only be used with the marker/shadow brush, the css tool allows you set images and gradients as your sketch background

  • copy the image address from the web and paste it in the css tool to set the bg
  • use the format 'gradient("corolstart","colorend")' to set a gradient or plain bg
  • type clear to clear the bg

shortcuts-- alt+t - set thickness, alt+l - lock bar, tap bar to unlock

Tip 3: notes

rightclick a note to delete it, note contents are automatically saved except when in handwriting mode, you can convert your written text to a note or edit notes with your handwriting

Tip 4: calibration

the calibration gives an overview of your pen precision and accuracy overtime
